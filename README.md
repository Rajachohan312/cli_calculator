# CLI Calculator

A simple, robust Command-Line Interface calculator built with Python. This tool allows users to perform quick arithmetic operations directly from their terminal.

## 🚀 Features
* **Basic Arithmetic:** Addition, Subtraction, Multiplication, and Division.
* **Error Handling:** Graceful management of division by zero and invalid inputs.
* **Persistent Session:** Perform multiple calculations without restarting the script.
* **Clean UI:** Simple prompts and formatted output for readability.
